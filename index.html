🧪 Mini Projeler (Temel → Orta)

1) Sayı Tahmin Oyunu

Rastgele 1–100 sayı; kullanıcı tahmin eder, ipucu ver.

import random
hedef = random.randint(1, 100)
tahmin = 0
hak = 7
while hak > 0:
    tahmin = int(input("Tahmin: "))
    if tahmin == hedef:
        print("Bildin!")
        break
    print("Yukarı" if tahmin < hedef else "Aşağı")
    hak -= 1
print("Sayı:", hedef)

2) To-Do (Dosyaya Kaydeden)

def ekle(m): 
    with open("todo.txt","a",encoding="utf-8") as f: f.write(m+"\n")
def liste():
    try:
        print(open("todo.txt","r",encoding="utf-8").read())
    except FileNotFoundError:
        print("Liste boş")
while True:
    komut = input("(e)kle, (l)iste, (q)uit: ")
    if komut == "e": ekle(input("Madde: "))
    elif komut == "l": liste()
    else: break

3) Web Scraping (requests + bs4)

import requests
from bs4 import BeautifulSoup
html = requests.get("https://example.com").text
soup = BeautifulSoup(html, "html.parser")
for a in soup.select("a"):
    print(a.get_text(strip=True), a.get("href"))

Not: Bazı siteler scraping'e izin vermez; robots.txt & hukuka dikkat.
